How to run poker script:

1. Download the file 'poker.py', go to that directory, and put the input file in the same directory.
2. In the command line type 'python poker.py <txt input filename>'.

Ex: If the poker hand file is called 'poker-hands.txt' and it is in the same directory as the 'poker.py' file, the command to run the program is:
	'python poker.py poker-hands.txt'

Outputs will be printed to the command line.
